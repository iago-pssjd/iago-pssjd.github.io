        # Short summary of my Data Scientist Experience

        During my first year of experience I worked as a programmer for the ATHLOS project (opens new window), harmonising and codifying the algorithms to produce a unique harmonised dataset from the databases of 18 distinct epidemiological studies. Further, one html report was built with RMarkdown from each study-variable harmonisation (the codes (opens new window)), where pre- and post- harmonised data was visualized and validated, and harmonisations codes were shown.

        The next years I started to analyse epidemiological data from the Global Burden of Disease project (GBD (opens new window)) with distinct statistical analysis technics as regression models for longitudinal data and also with psychometrics and machine learning techniques for the identification and reduction of the dimensionality of the data.

        Last year, among other projects, I started to process and analyse Fitbit data using longitudinal machine learning technique for the RADAR-CNS project (opens new window). The big volume of data handled posed a challenge and the obligation to automate many tasks as file handling through Bash scripts in Debian.


        *Programmer or Data Scientist are directions in which I would like to pursue my career. Having some experience with different programming languages (beyond R, with C during my degree, and with Python making courses from distinct fields on my own and using from time to time depending on the needs) and big datasets allows me to have a good insight and a more in-depth understanding of underlying issues that can be faced when managing a project.
